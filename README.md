# Doctor Appointments Frontend

Scaffolded with Vite + React + TypeScript + Tailwind CSS + MSW (Mock Service Worker).

## Scripts
- npm run dev - Start dev server
- npm run build - Build production
- npm run preview - Preview build

## Notes
- MSW is setup under `src/mocks` and is registered in `src/main.tsx` during development.
- Run `npm install` to install deps.
